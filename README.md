# -Atividade-Programa-o-em-Java
 Este repositório contém a implementação de sete programas em Java desenvolvidos para praticar entrada de dados, condições, loops, classes e operações matemáticas.
## Tecnologias Utilizadas
- Java 8+
- Scanner
- JOptionPane
- Terminal ou IDE (Eclipse, NetBeans, IntelliJ, VS Code)

---

# Tarefas Implementadas

## 1. **Calculadora de Desconto**
Recebe o valor de um produto e a porcentagem de desconto. 
Calcula:
- valor do desconto 
- preço final 

Entrada via **Scanner**.

---

## 2. **Calculadora de ITBI**
Recebe:
- valor de transação
- valor venal
- porcentagem do ITBI 

O imposto é calculado sobre **o maior valor**. 
Entrada via **JOptionPane**.

---

## 3. **Sistema de Avaliação**
Recebe duas notas de prova e um trabalho. 
Calcula a média e exibe se o aluno está **aprovado** (>=6) ou **reprovado**. 
Entrada via **JOptionPane**.

---

## 4. **Calculadora de Aposentadoria**
Recebe idade, sexo e anos de contribuição. 
Determina:
- se pode se aposentar 
- ou quantos anos faltam por idade e por contribuição 

Regras utilizadas:
- Homem: 65 anos ou 35 anos de contribuição 
- Mulher: 62 anos ou 30 anos de contribuição 

Entrada via **Scanner**.

---

## 5. **Circuito de Resistências**
Recebe quatro resistências em série. 
Calcula:
- resistência equivalente 
- maior resistência 
- menor resistência 

Entrada via **Scanner**.

---

## 6. **Sistema de Login**
Usuário tem **3 tentativas** para entrar com login e senha "java8". 
Exibe mensagens de erro e bloqueia após esgotar.

Entrada via **Scanner**.

---

## 7. **Gerador de Tabuada**
Recebe um número e exibe a tabuada de 1 a 10. 
Entrada via **Scanner**.
